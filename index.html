**Connected** * * * * * This project aims at experimenting interaction between virtual and real worlds through connected objects. There are 2 parts : - a LUA mod to be used in [Minetest game](http://www.minetest.net/) - a [nodejs](https://nodejs.org/) application to serve as a bridge between Minetest and connected objects.
